# Examples

These are sample outputs from one free run of the skill, for a fictional invoicing company (acme-invoicing.com). Your own run produces files like these for your real site.

- **dashboard.html** is the deliverable. Open it in a browser (double-click, or `file://` it). It holds the site audit, a sortable keyword table with a CSV export button, the 30-day content calendar, and the sample article preview, plus the next step. The skill uses this exact file as its template and swaps in your data, so the styling stays consistent.
- **sample-article.md** is the full article the free skill writes (one finished, fact-checked, publish-ready piece). It shows the quality bar.
- **keywords.csv** is the export the dashboard's "Export CSV" button produces.

The data here is illustrative. Volumes are labeled as estimates because the free tier does not use a paid data provider. Real measured volume and difficulty come with [The SEO Agent](https://www.theseoagent.ai).
